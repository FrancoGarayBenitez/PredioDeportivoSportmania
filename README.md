# Sportmania 

---

***Sportmania*** es un sitio web estático realizado para egresarme en el curso de Desarrollo Web de CoderHouse.

Trata acerca de un predio deportivo que presenta su gimnasio, alquiler de canchas de fútbol y pádel, además de otros servicios.

---

#### Instalación 📋

Para visualizar el sitio solo se precisa de clonar el repositorio y luego ejecutar en el navegador el archivo index.html.

Para iniciar el preprocesador SASS hay que instalarlo con `npm install` en consola y luego ejecutar el comando `npm run watch-css`.
